Data Security

We use Firebase Authentication to securely manage user logins and Firebase Firestore to store messages and user data. Both services are provided by Google and follow strong security standards, including encryption at rest and in transit. Access to data is controlled via Firebase Security Rules, allowing only authenticated users to access their own data. While this is a non-commercial project, best practices in data handling and security are applied where possible.

Storage and Cookies

This application uses Firebase Authentication and Firestore, which may utilize browser storage mechanisms such as cookies, local storage, or IndexedDB. These technologies are used exclusively to maintain secure user sessions, enable login functionality, and support real-time messaging. No cookies are used for tracking or marketing purposes. For more information on how Firebase handles data, please refer to the Firebase Privacy Policy: Firebase Privacy Policy.
